What you need to know to build a website

Building a website requires 3 languages: HTML, CSS, Javascript.
What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is used to display headings, text, buttons, list, images, or links to a website.
  • HTML follows a certain grammar and uses heading tags and closing tags to display headings, text, buttons, list, images or links.
    • Most Frequently used HTML tags
      Tells your browser that you will be using the HTML language
    1. "< html> < /html>
      2. Tells your browser that there is a head section. This is added before the body.
    2. "< head> < /head>"
      3. Tells your browser that there is a body section.
    3. "< body> < /body>"
      4. Tells your browser that there is heading1, heading2,... heading6. Heading1 is the largest and Heading6 is the smallest
    4. "< h1> This is where your heading goes < /h1>"
    5. "< h2> This is where your heading goes < /h2>"
    6. "< h3> This is where your heading goes < /h3>"
    7. "< h6> This is where your heading goes < /h6>"
      8. Tells your browser that there is a paragraph.
    8. "< p> This is where your paragraph goes < /p>"
      9. Tells your browser that there is an image.
    9. "< img src="Your-Image-File.jpeg">
      10. Tells your browser that there is a section in your HTML code.
    10. "< div> "
      11. Tells your browser that there is an anchor tag. Anchor tag are used for a link destination.
    11. "< a href="https://www.google.com"> < /a>"
      12. Href means hyperlink reference. The href attributes specifies the URL of where the link will go.

      Keep in mind that there is more HTML tags. However, this is the basics you will need to make a website.

What is CSS?

  • CSS stands for Cascading Style Sheet
  • CSS is used to style and customize your website by adding colors, animation, and fonts.
  • CSS follows a certain grammar that uses classes and id to customize HTML tags

    • A few examples written in HTML and CSS
  • "< p class="black-text"> This is where your paragraph goes < /p>"
    "< style>
    .black-text {
    color: black;
    }
    < /style>"

What is Javascript?

  • Javascript is a high level programming language that is used as a client side scripting language
  • Javascript code is written into HTML pages.
  • Javascript can be used to make an change or an action to your website. For example, clicking on an image will require Javascript to open up another HTML page.

The life of a software developer

First off, if you are a software developer or learning how to code. Give yourself a pat in the back. Seriously.
You are awesome!
Don't ever forget that!
So, what is a software developer?
A software developer is someone who creates computer software including the design and testing of a software such as mobile apps, desktop apps, or websites.
An example will be this website that I designed and created for you.
Being a software developer is a talented skill to have and requires a shit ton of patience and perseverance. It also requires you to be detail oriented and have a passion for technology.
I also want to mention that at some point in your life you might be discouraged to learn how to code or have self doubt in your software career.
This is normal. As a matter of fact many programmers face imposter syndrome at some point in his or her life. So, please do not be discouraged.
You are never too late to start learning how to code.
Just remember nobody in this world cannot take away your knowledge.
Lastly, tech is changing rapidly and many programmers must upgrade his or her skills in order to stay competitive and up to date in the software industry.

Why you should fail more often

The faster you fail the higher chance you will succeed.

Let's be honest. Failing sucks! But you can learn from your failures and enable yourself to think wisely and intelligently. Failures applies to everything in life not just school.
Learn to embrace failure.
This will take some time to get use to because as a human beings we tend to fear change. We tend to avoid change. We fear rejection.
In simple terms, you cannot be successful or want to be successful without taking the failures.
You must learn to love failure.
You must love eating the dirt in order to reach your end goal.
Do not take failure as a pessimistic view and give up because of it.
